# AI Document Intelligence Assistant

AI-powered document analysis platform built using Python, Streamlit, FAISS, and Groq LLM.

# Live Demo
https://ai-document-intelligence-assistant-atsyfsskfp5gwjr3gogsrx.streamlit.app/

## Features
- Research Paper Q&A Chatbot
- Research Gap Finder
- Resume Analyzer
- Report Summary Generator
- Key Findings Extraction

## Tech Stack
- Python
- Streamlit
- NumPy
- FAISS
- Sentence Transformers
- Groq API

## Workflow
1. Upload PDF document
2. Extract text from PDF
3. Convert text into embeddings
4. Store vectors using FAISS
5. Generate AI-powered insights

## Installation
```bash
pip install -r requirements.txt
streamlit run app.py
```
